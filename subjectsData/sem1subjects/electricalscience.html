  • 4. Finding Roots of f(x)=0 using Newton Raphson Method

    Overview

    This experiment uses the Newton-Raphson method, a numerical technique for finding roots of nonlinear equations.

    Procedure

    1. Define the function f(x) and its derivative f’(x).
    2. Choose an initial guess for the root.
    3. Iterate using the formula x = x - f(x)/f’(x) until convergence.

    Notes

    - Ensure derivative function is correct for accurate convergence.
    - Choose an initial guess close to the expected root.
